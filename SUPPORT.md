# Support

For bugs and feature requests, use GitHub Issues:

https://github.com/jordachmakaya/env-sync/issues

Before opening an issue:

- Search existing issues.
- Include the package version.
- Say whether the problem is with GitHub, GitLab, or shared CLI behavior.
- Include sanitized logs only. Do not share secrets, tokens, full `.env` files, private project IDs, or private URLs.

For security reports, use GitHub Security Advisories instead of public issues:

https://github.com/jordachmakaya/env-sync/security/advisories/new
